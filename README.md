## 📝 图片格式建议

- **格式**: JPG 或 PNG（JPG 适用于照片，PNG 适用于带透明度的图形）
- **质量**: 80-90%（平衡文件大小和清晰度）
- **文件大小**: 每张图片建议控制在 500KB 以内，以确保网页加载速度
- **色彩模式**: RGB（用于网页显示）

---

## 🎨 设计风格提示

为了保持网站的现代科技感蓝色主题：

1. **Banner 图片**: 建议使用蓝色调或冷色调的科技感背景
2. **产品图片**: 使用白色或浅灰色纯色背景，突出产品细节
3. **证书图片**: 确保文字清晰可读，建议扫描分辨率 300dpi 以上
4. **公司图片**: 展现专业、现代化的企业形象

---

## ⚠️ 注意事项

1. **文件名必须完全匹配**: 请确保文件名（包括大小写和空格）与上述清单完全一致
2. **图片路径**: 所有 HTML 文件已配置为从 `images/` 目录加载图片
3. **备用方案**: 如果图片暂时未准备好，网站会显示备用占位符，但建议尽快替换为实际图片
