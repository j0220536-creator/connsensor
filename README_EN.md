## 📝 Image Format Recommendations

- **Format**: JPG or PNG (JPG for photos, PNG for graphics with transparency)
- **Quality**: 80-90% (balance between file size and clarity)
- **File Size**: Keep each image under 500KB to ensure fast page loading
- **Color Mode**: RGB (for web display)

---

## 🎨 Design Style Guidelines

To maintain the modern tech blue theme of the website:

1. **Banner Images**: Use blue-toned or cool-toned technology-style backgrounds
2. **Product Images**: Use white or light gray solid backgrounds to highlight product details
3. **Certificate Images**: Ensure text is clear and readable; recommended scanning resolution of 300dpi or higher
4. **Company Images**: Present a professional and modern corporate image

---

## ⚠️ Important Notes

1. **Filenames must match exactly**: Ensure filenames (including capitalization and spaces) match the list above exactly
2. **Image paths**: All HTML files are configured to load images from the `images/` directory
3. **Fallback option**: If images are not ready yet, the website will display placeholder elements, but we recommend replacing them with actual images as soon as possible
